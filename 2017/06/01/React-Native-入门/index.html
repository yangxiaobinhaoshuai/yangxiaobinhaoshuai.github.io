React Native 入门

发表于 2017-06-01

  网上的入门资料一百度一大把，随随便便一篇边看边敲也得个小半天,我更喜欢先会写个大概，在慢慢理解，看了中文官网教程，很详细，有原理有实例。

编辑器

  编辑器上我比较喜欢JetBrains 的WebStorm ，因为Android Studio 用惯了哈… 建议下载非社区版的，百度激活码有很多，直接新建project 就支持RN的project ，觉得用命令行建工程怪怪的..

运行Android平台 报错
  1. 8081端口占用的话，看看是否还用着命令行运行着工程，退掉就好，实在不行换一个端口运行
  2. sdk.dir…local.properties…ANDROID_HOME… 这个错误是因为RN工程没有Loacl.properties这个配置文件，直接从sutdio 工程中copy一个过来即可
  3. 运行后报“/bin/sh: adb: command not found” 无论怎么配置环境变量总是需要source 一下才能生效，搞了一下午啊，脑袋晕晕的。最后找到一篇帖子成功解决问题，终于在手机上运行了helloworld！ 差点下不了班..shit!
运行iOS平台报错
  1. 需要安装xCode
  2. 配置看这篇帖子
  3. xcrun: error: unable to find utility “instruments”, not a developer tool or in PATH 解决办法
语法 js+react
  1. js：RN从0.14版本后就一直用ES6的语法，所以建议我们读一读阮一峰老师的书，感谢阮老师
